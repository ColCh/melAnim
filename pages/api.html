Проект экспортируется в глобальное пространство имён под именем melAnim, что является ссылкой на функцию animate

animate принимает несколько аргументов 

        animate (elements, properties, duration, easing, complete)
Где:
  • elements — Элемент(ы) для анимирования. Элемент, массив элементов, или список элементов (NodeList)
  • properties — Свойства для анимирования в формате jQuery.animate (Ключ - имя свойства, значение - конечная величина свойства).
  • duration — Продолжительность анимаци. Формат: число (миллисекунды) или CSSTimeString (число + размерность - секунды s или миллисекунды ms)
  • easing — Смягчение всей анимации (алиас, CSS Timefunction, аргументы к временной функции CSS или сама JS функция)
  • complete — Обработчик события завершения анимации. Исполнится, когда анимация проиграется до конца.
