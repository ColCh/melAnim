Проект экспортируется в глобальное пространство имён под именем вызываемого объекта melAnim, что является ссылкой на функцию animate Структура вызываемого объекта melAnim:

Animate
объект-builder для анимаций на JS и CSS

animate принимает несколько форматов аргументов 

animate (elements, properties[, duration][, easing][, complete])
Где:
elements
Элемент(ы) для анимирования. Элемент, массив элементов, или список элементов (NodeList)
properties
Свойства для анимирования в формате jQuery.animate (Ключ - имя свойства, значение - конечная величина свойства).
duration
Продолжительность анимаци. Формат: число (миллисекунды) или CSSTimeString (число + размерность - секунды s или миллисекунды ms)
easing
Смягчение всей анимации (алиас, CSS Timefunction, аргументы к временной функции CSS или сама JS функция)
complete
Обработчик события завершения анимации. Исполнится, когда анимация проиграется до конца.

Animate — функция-строитель для двух классов анимаций : JS и CSS. При передачи аргументов выберет нужный режим, основанный на статусе поддержки CSS3 анимаций текущим браузером, или на основе переданных аргументов, и вернёт созданный экземпляр, настроенный по переданным аргументам - настроить объект можно и напрямую. Имеет 2 формата передачи аргументов. Первый имеет, в целом, такой же формат, что и у animate, за исключением формата передачи свойств. 

Animation (elements, keyframes, duration, easing, oncomplete)
Где:
elements
Элемент(ы) для анимирования. Элемент, массив элементов, или список элементов (NodeList)
keyframes
Свойства для анимирования в формате CSS Keyframes (обратитесь в документацию по CSS3 анимациям).
duration
Продолжительность анимаци. Формат: число (миллисекунды) или CSSTimeString (число + размерность - секунды s или миллисекунды ms)
easing
Смягчение всей анимации (алиас, CSS Timefunction, аргументы к временной функции CSS или сама JS функция)
complete
Обработчик события завершения анимации. Исполнится, когда анимация проиграется до конца.
Второй вид формата аргументов сжимает их визуальное количество в объект, но при этом можно передать немногим больше аргументов. 
Animation (elements, keyframes, options)
Где:
elements
Элемент(ы) для анимирования. Элемент, массив элементов, или список элементов (NodeList)
properties
Свойства для анимирования в формате jQuery.animate (Ключ - имя свойства, значение - конечная величина свойства).
options
Объект с настройками и параметрами.
Структура объекта options — обычный plain Object:
classicMode
Bool. Используется ли классический режим (true), или режим CSS3 анимаций (false).
easing
STRING|ARRAY|FUNCTION. Смягчение анимации при прогрессе или всей анимации. Возможно установить особое смягчение для свойства (на протяжении всей анимации). Установленное смягчение будет использовано, если прогресс по проходу будет соответствовать неравенству: ТЕКУЩИЙ_КЛЮЧЕВОЙ_КАДР <= ПРОГРЕСС_ПО_ПРОХОДУ < СЛЕДУЮЩИЙ_КЛЮЧЕВОЙ_КАДР. Метод устанавливает смягчение для "текущего" (см. неравенство) ключевого кадра. При установке смягчения для свойства параметр прогресса игнорируется. (!) Абсциссы первой и второй точек для кубической кривой должны принадлежать промежутку [0, 1]. Установка JS-функции как смягчения возможна только в классическом режиме.
duration
TIME. Продолжительность прохода анимации. Отрицательные значения считаются за нулевые. Нулевое значение соответствует мгновенному проходу анимации, при этом все события (конца прохода и конца анимации) возникают так же, как и при положительной продолжительности прохода, а так же и режим заполнения (fillMode) работает так же, как и при положительной продолжительности прохода.
direction
STRING. Направление анимации Значение "normal" соответствует возрастанию прогресса от 0 до 1 при каждом проходе. Значение "reverse" соответствует убыванию прогресса от 1 до 0 при каждом проходе. Значение "alternate" соответствует направлению "normal" для нечётных проходов и "reverse" для чётных. Значение "alternate-reverse" соответствует направлению "reverse" для нечётных проходов и "normal" для чётных.
iterationCount
NUBMER. Число проходов цикла анимации. Значение "infinite" соответствует бесконечному числу повторений анимации. Дробные значения соответствуют конечному значению прогресса по проходу. Отрицательные числовые значения игнорируются.
delay
TIME. Задержка старта Если значение положительное, старт анимации будет отложен на численное представление аргумента. Если отрицательное, то при старте будет считаться, что прошло уже указанное временя с момента старта.
fillMode
STRING. Режим заполнения. Значение "backwards" соответствует отрисовке значений начального ключевого кадра сразу после старта (и перед самим анимированием). Значение "forwards" соответствует отрисовке значений конечного ключевого кадра после окончания анимации. Значение "none" не соответствует ни одному из значений; Значение "both" соответствует и первому, и второму одновременно.
onstart
FUNCTION. Обработчик события старта анимации - исполнится, когда анимация начнет проигрываться
oniteration
FUNCTION. Обработчик события завершения прохода. Функция не вызывается, если конец прохода является концом анимации.
oncomplete
FUNCTION. Обработчик события завершения анимации.
onstep
FUNCTION. Обработчик события ступени анимации. Будет исполняться на каждом шаге анимации. Установка возможна только для JS-анимаций.
